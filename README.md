# cotizacion_credito_backend

## API REST 

API que funciona como sistema de cotización de créditos, en base a un producto seleccionado.

La API tiene distintos endpoints:

* Credit
  * createQuote
  * getQuote
* Deadline
  * createDeadline
  * getDeadlines
* Product
  * createProduct
  * getProducts
  * patchProduct
  * deleteProduct
